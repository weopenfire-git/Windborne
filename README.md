# FlightLog 飞行日志

一个炫酷的驾驶舱 HUD 风格飞行记录网站，基于纯静态 HTML + CSS + JS，所有数据存储在浏览器本地（localStorage）。

---

## 功能概览

| 功能 | 说明 |
|------|------|
| **个人主页** | 显示头像、昵称、注册号、飞行统计仪表盘 |
| **航程记录** | 完整复刻纸质飞行日志的所有字段（飞行信息、离场/到达、留言） |
| **机票上传** | 支持上传登机牌、购物小票、糖果纸等照片（base64 本地存储） |
| **飞行地图** | SVG 世界地图 + 航线标记 + 动画飞机航线 |
| **数据统计** | 总飞行次数、距离、时长、机型分布、航空公司分布 |
| **数据导出** | 支持 JSON 格式导出/导入，防止数据丢失 |

---

## 设计风格

- **OLED 深色背景** `#0B0B10` + 科技蓝 `#3B82F6`
- **HUD 驾驶舱风格**：扫描线、网格背景、玻璃拟态、发光描边
- **字体**：Orbitron（标题）+ Exo + Roboto Mono（数据）+ Noto Sans SC（中文）
- **动画**：星空背景、数字滚动、飞机悬浮、航线流动动画

---

## 字段说明（基于纸质版）

### 飞行信息
- 注册号、飞行日期、航班号、起飞机场（ICAO）、目的机场（ICAO）
- 机型、注册号、航空公司、座位、座位位置、客舱等级

### 航行参数
- V1 / Vr / V2、飞行层级、巡航高度、巡航速度（CAS）
- 经停、到达地、飞行目的

### 离场 / 到达信息
- 航站楼、登机口、跑道、停机位、预计/实际起飞时间
- 抵达航站楼、着陆跑道、实际到达时间、Vref/VLS、进场程序

### 留言
- 乘客留言、机长留言、副机长留言、乘务长留言

### 票据粘贴处
- 上传登机凭证、购物小票、糖果纸等照片

## 使用方式

### 方式一：直接打开
1. 在浏览器中打开 `index.html` 即可使用
2. 所有数据自动保存在浏览器 localStorage 中

### 方式二：部署到网页
1. 将 `flight-log/` 目录中的文件上传到任意静态托管服务（如 GitHub Pages、Vercel、Netlify、Cloudflare Pages）
2. 无需后端，无需数据库

---

## 数据备份与迁移

在「设置」页面：
- **导出**：点击「导出数据 (JSON)」将数据下载到本地
- **导入**：点击「导入数据」选择之前导出的 JSON 文件
- **清空**：点击「清空所有数据」可重置（⚠️ 不可逆）

> **提示**：建议定期导出数据备份，更换浏览器或设备时可通过导入迁移数据。

## 技术栈

- 纯 HTML + CSS + JavaScript（零依赖，零构建）
- localStorage 本地持久化
- Google Fonts 字体加载
- 纯 CSS 动画（无需 JS 动画库）
