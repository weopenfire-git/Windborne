# Windborne · 飞行日志社区

> 飞友的个人飞行日志 + 动态广场 + 知识库

## 两个版本

### v1 · flight-log（已上线）

纯前端飞行日志，数据存在浏览器 localStorage。

- **在线访问**：https://weopenfire-git.github.io/Windborne/
- **文件**：`index.html` + `styles.css` + `app.js`
- **功能**：航程记录、机票上传、飞行地图、统计仪表盘、数据导入导出
- **状态**：✅ 已上线，儿子和朋友可直接使用

### v2 · Windborne 社区（开发中）

从 v1 升级为全栈社区应用，三层产品形态：

```
┌─────────────────────────────────────────┐
│  ③ 知识库 · 机型/机场/航司百科            │
├─────────────────────────────────────────┤
│  ② 动态广场 · 飞行报告/拍机/点评/问答     │
├─────────────────────────────────────────┤
│  ① 个人飞行日志 · 航程/机票/统计/地图     │
└─────────────────────────────────────────┘
```

- **技术栈**：Next.js 14 + TypeScript + Supabase + Cloudflare Pages
- **状态**：📝 规划完成，模块 1（数据库）代码已就绪，待 Supabase 注册后执行迁移
